# DirectoryApp

Simple Project for storing information about students. Aimed for learning purposes.

## Used Technologies

```
Spring Boot
Spring MVC
H2-database
Lombok Project
Spring Jpa
JSP

```

## Installation

```bash
git clone [url]
mvn spring-boot:run
```

## Usage

```
https://localhost:port/{requestName}

Use to list students:
https://localhost:port/studentDirectory 

```
